# Installation

## Requirements

- Node.js 22 or later
- npm, pnpm, or yarn

## Package Managers

### pnpm (recommended)

```bash
pnpm add -D @marlinjai/clearify
```

### npm

```bash
npm install @marlinjai/clearify --save-dev
```

### yarn

```bash
yarn add -D @marlinjai/clearify
```
