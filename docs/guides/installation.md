# Installation

## Requirements

- Node.js 20 or later
- npm, pnpm, or yarn

## Package Managers

### npm

```bash
npm install clearify --save-dev
```

### pnpm

```bash
pnpm add -D clearify
```

### yarn

```bash
yarn add -D clearify
```
